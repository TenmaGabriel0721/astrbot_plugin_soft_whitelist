# astrbot_plugin_soft_whitelist

高优先级软白名单插件。

## 当前策略
- 不处理 `request / notice / meta_event`，避免影响好友申请、群邀请和其他管理插件
- `message` 按白名单放行，其余拦截
- 非白名单的好友私聊 / 临时会话可自动回复
- 自动回复仅在收到对方消息时触发，且按对方QQ每天最多一次
- AstrBot 主配置里的 `admins_id` 可无视白名单
- 可通过管理员命令清理 Bot 已退出群聊的群白名单记录

## 管理指令
- `~软白名单状态`：查看开关、白名单列表和自动回复配置
- `~加群白 群号` / `~删群白 群号`：管理群聊白名单
- `~加群员白 QQ号` / `~删群员白 QQ号`：管理群聊成员白名单
- `~加好友白 QQ号` / `~删好友白 QQ号`：管理好友私聊白名单
- `~加临时白 用户 QQ号` / `~删临时白 用户 QQ号`：管理临时会话用户白名单
- `~加临时白 群 群号` / `~删临时白 群 群号`：管理临时会话来源群白名单
- `~好友白回复 开|关` / `~好友白回复 设定 回复内容`：管理好友非白名单自动回复
- `~临时白回复 开|关` / `~临时白回复 设定 回复内容`：管理临时会话非白名单自动回复
- `~清理退群白`：剔除群聊白名单和临时来源群白名单中 Bot 已退出的群；若平台返回 0 个当前群但本地仍有记录，会中止以避免误删
- `~清理退群白 强制`：确认 Bot 确实不在任何群时，强制按当前平台返回结果清理
